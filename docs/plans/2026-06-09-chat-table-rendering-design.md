# Design Document: Chat Table Rendering

## Overview
This document specifies the design for rendering Markdown tables in the AI chat sidebar of RocketMap.

## Requirements
- Standard Markdown tables generated by AI must render correctly as HTML tables.
- Table rendering must work during streaming.
- Styling must follow the "Calm Until Critical" guidelines: minimalist, dark/transparent background, subtle borders, matching typography.

## Proposed Design (Approach A)

### 1. Package Installation
Add `remark-gfm` (version 4) to support GitHub Flavored Markdown (specifically tables) in React 19 and react-markdown v10.
```bash
npm install remark-gfm@4
```

### 2. Integration
Import and apply `remarkGfm` plugin to `<ReactMarkdown>` components inside `ChatMessage.tsx`:
```tsx
import remarkGfm from "remark-gfm";

// applied as:
<ReactMarkdown remarkPlugins={[remarkGfm]}>{content}</ReactMarkdown>
```

### 3. CSS Styling
Add classes in `app/globals.css` under the `.chat-markdown` selector:
```css
.chat-markdown table {
  width: 100%;
  border-collapse: collapse;
  margin: 0.5rem 0;
  font-size: 0.75rem;
  line-height: 1.25rem;
}

.chat-markdown th {
  border-bottom: 1px solid rgba(255, 255, 255, 0.1);
  padding: 0.375rem 0.5rem;
  font-weight: 600;
  text-align: left;
  color: rgba(255, 255, 255, 0.9);
  background-color: rgba(255, 255, 255, 0.02);
}

.chat-markdown td {
  border-bottom: 1px solid rgba(255, 255, 255, 0.05);
  padding: 0.375rem 0.5rem;
  color: rgba(255, 255, 255, 0.7);
}

.chat-markdown tr:hover {
  background-color: rgba(255, 255, 255, 0.02);
}
```
